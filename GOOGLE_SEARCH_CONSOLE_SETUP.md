# Google Search Console Setup

## Steps to verify your site:

1. Go to: https://search.google.com/search-console
2. Click "Add Property"
3. Enter: `https://aminch18.github.io/portfolio`
4. Choose "HTML tag" verification method
5. Copy the verification code (looks like: `google-site-verification=ABC123XYZ...`)
6. Add it to `src/app/layout.tsx` in the metadata section (already prepared)
7. Deploy your site
8. Go back to Search Console and click "Verify"

## After Verification:

### Submit Sitemap:
1. In Search Console, go to "Sitemaps" in the left menu
2. Enter: `sitemap.xml`
3. Click "Submit"

### Monitor Performance:
- Check "Performance" tab to see search queries
- Monitor "Coverage" for indexation status
- Review "Enhancements" for any issues
